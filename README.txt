CAMERA CONTROLS
--------------------------------------------------
C				: Toggle freelook

W/S				: Move forward/back
A/D				: Move left/right
SPACE/LSHIFT	: Move up/down

LEFT/RIGHT		: Look left/right
UP/DOWN			: Look up/down

+/-				: Zoom in/out

[/]				: Far plane closer/farther
;/'				: Near plane closer/farther

BACKSPACE		: Reset zoom, near/far planes
--------------------------------------------------
